Section I

Current System: The Identical Circuit Problem

At present, every agent a ∈ {xs, s, m, l, xl, xxl, 5m, 10m} passes through the same quantum circuit construction at _create_quantum_circuit() (L:10706). The circuit operates on n = 6 qubits with the following three stages:

1.1   State Encoding (Rotation Layer)

The classical state vector s ∈ ℝ⁵⁸ is normalized into rotation angles θ ∈ [0, 2π]⁶ and applied as single-qubit Y-rotations:

Definition 1 — State Encoding
$$R_{\text{encode}} = \bigotimes_{i=0}^{5} R_Y(\theta_i) \qquad \text{where} \quad \theta_i = \frac{s_i - s_{\min}}{s_{\max} - s_{\min} + \epsilon} \cdot 2\pi$$
$$|\psi_0\rangle = R_{\text{encode}} \, |000000\rangle$$

1.2   Entanglement Layer (CNOT Chain)

A linear chain of CNOT gates entangles adjacent qubits. This is the same for every agent:

Current — Identical for ALL 8 Agents
$$U_{\text{entangle}} = \text{CNOT}_{4,5} \cdot \text{CNOT}_{3,4} \cdot \text{CNOT}_{2,3} \cdot \text{CNOT}_{1,2} \cdot \text{CNOT}_{0,1}$$
$$|\psi_1\rangle = U_{\text{entangle}} \, |\psi_0\rangle$$

In graph notation, this is the topology G = {(0→1), (1→2), (2→3), (3→4), (4→5)} — a simple path graph. Every agent uses this exact graph.

1.3   Phase Encoding Layer

A second layer applies Z-rotations with offset indices for additional expressivity:

Definition 2 — Phase Encoding
$$R_{\text{phase}} = \bigotimes_{i=0}^{5} R_Z\!\left(\theta_{(i + 3) \bmod 58}\right) \qquad |\psi_2\rangle = R_{\text{phase}} \, |\psi_1\rangle$$

1.4   Measurement & Feature Extraction

The statevector simulator computes the exact final state, and features are the Born-rule probabilities:

Definition 3 — Quantum Feature Vector
$$\mathbf{f}_q = \Big[ \, |\langle k \,|\, \psi_2 \rangle|^2 \Big]_{k=0}^{63} \in \mathbb{R}^{64}$$

where |k⟩ is the k-th computational basis state of the 6-qubit Hilbert space ℋ = (ℂ²)⊗⁶

The Problem
Since Uentangle is identical for all agents, and the state s is the same market snapshot: $$\mathbf{f}_q^{(\text{xs})} = \mathbf{f}_q^{(\text{s})} = \mathbf{f}_q^{(\text{m})} = \cdots = \mathbf{f}_q^{(\text{10m})}$$ All 8 agents receive the same quantum features, producing the same Q-values, producing the same votes, producing a permanent lock to SELL for 2+ hours.

Proposed Solution: Agent-Specific Entanglement Topologies

We replace the single universal entanglement operator Uentangle with a family of eight distinct operators, one per agent. Each operator defines a unique directed graph Ga = (V, Ea) over the 6 qubit vertices, determining which features correlate with which.

Proposal — Per-Agent Entanglement
$$U_{\text{entangle}}^{(a)} = \prod_{(c,t) \,\in\, E_a} \text{CNOT}_{c,t} \qquad \forall \; a \in \mathcal{A}$$

where 𝒜 = {xs, s, m, l, xl, xxl, 5m, 10m} and Ea is the unique edge set for agent a

The full circuit for agent a becomes:

Modified Full Circuit — Agent a
$$|\psi_{\text{final}}^{(a)}\rangle = R_{\text{phase}} \cdot U_{\text{entangle}}^{(a)} \cdot R_{\text{encode}} \; |000000\rangle$$
$$\mathbf{f}_q^{(a)} = \Big[ \, |\langle k \,|\, \psi_{\text{final}}^{(a)} \rangle|^2 \Big]_{k=0}^{63}$$
Key Property
Since Uentangle(xs) ≠ Uentangle(5m), even given the same input state, the outputs are structurally different: $$\mathbf{f}_q^{(a)} \neq \mathbf{f}_q^{(b)} \quad \forall \; a \neq b$$ This is not randomness — it is genuine structural diversity. Different CNOT patterns extract different multi-qubit correlations from the same encoded state.

Section IV

Mathematical Proof of Feature Diversity

4.1   Why Different CNOT Topologies Produce Different Outputs

The CNOT gate acting on control qubit c and target qubit t is the unitary:

Definition 4 — CNOT Operator
$$\text{CNOT}_{c,t} = |0\rangle\langle 0|_c \otimes I_t \;+\; |1\rangle\langle 1|_c \otimes X_t$$

It flips the target qubit if and only if the control qubit is |1⟩. This creates entanglement between the two qubits — their states become correlated.

Crucially, CNOT gates do not commute in general when they share qubits:

Theorem 1 — Non-Commutativity
$$\text{CNOT}_{0,1} \cdot \text{CNOT}_{1,2} \;\neq\; \text{CNOT}_{1,2} \cdot \text{CNOT}_{0,1}$$

Therefore: different orderings of the same CNOT set produce different unitaries, and entirely different CNOT sets produce even more different unitaries.

4.2   Correlation Structure per Topology

Each entanglement graph Ga defines a specific correlation matrix over the 6 qubits. Let ρij(a) denote the two-qubit reduced density matrix of qubits i and j after applying topology a. The concurrence Cij(a) measures the entanglement between qubits i and j:

Correlation Structure
$$C_{ij}^{(a)} > 0 \quad \text{if and only if} \quad \text{qubits } i,j \text{ are connected (directly or transitively) in } G_a$$

Different graphs Ga → different non-zero entries in the correlation matrix → different feature vectors fq(a)

For example, in the linear chain (agent xs), qubits 0 and 5 are only weakly entangled (through 4 intermediaries). In the mirror topology (agent l), they are directly connected via CNOT0,5. This means:

Concrete Example
$$C_{0,5}^{(\text{xs})} \ll C_{0,5}^{(\text{l})} \qquad \text{and} \qquad C_{0,1}^{(\text{xs})} \gg C_{0,1}^{(\text{l})}$$

Agent xs is sensitive to local correlations (features 0↔1). Agent l is sensitive to global correlations (features 0↔5). They extract complementary information.

4.3   Feature Distance Guarantee

We can bound the minimum distance between any two agents' feature vectors. Let dTV denote total variation distance:

Theorem 2 — Minimum Diversity
$$d_{\text{TV}}\!\left(\mathbf{f}_q^{(a)}, \, \mathbf{f}_q^{(b)}\right) = \frac{1}{2} \sum_{k=0}^{63} \left| f_{q,k}^{(a)} - f_{q,k}^{(b)} \right| \;>\; 0 \quad \forall \; a \neq b$$

This holds for any non-trivial input state because U(a) ≠ U(b) as operators. The feature vectors are guaranteed to differ for every market snapshot — not by randomness, but by construction.

$$\mathbf{f}_q^{(a,t)} = \Big[ |\langle k | \psi_2^{(a,t)} \rangle|^2 \Big]_{k=0}^{63} \in \mathbb{R}^{64}$$

Step 6 — Q-Value Computation:

$$\mathbf{Q}^{(a,t)} = \text{DQN}_a\!\left(\text{fusion}\!\left(\text{classical}_a(\mathbf{s}^{(t)}), \; \mathbf{f}_q^{(a,t)}\right)\right) \in \mathbb{R}^2$$

$$\alpha^{(a,t)} = \begin{cases} \text{argmax}_j \; Q_j^{(a,t)} & \text{with probability } 1 - \epsilon_a \\ \text{Uniform}(\{0,1\}) & \text{with probability } \epsilon_a \end{cases}$$

$$\alpha_{\text{final}}^{(t)} = \text{AVN}\!\left(\{\mathbf{Q}^{(a,t)}\}_{a \in \mathcal{A}}, \; \{\alpha^{(a,t)}\}_{a \in \mathcal{A}}\right)$$

The only change from the current system is in Step 3. Steps 1, 2, and 4–8 remain identical. The modification is surgically precise: we replace one universal unitary with eight agent-specific unitaries.
